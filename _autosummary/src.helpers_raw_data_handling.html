src.helpers_raw_data_handling

This module handeles all the low level raw data such as:

  • reading from csv to dataframe

  • syncing diffrent measurements timestamps and validate how synced they are

  • checking measurement and validate the sampling rate and missing samples

  • physical unit conversions

  • save and read dataframes in/from light format ‘hdf’

3. myHDFRead(path,key=’a0’)- This function read dataframe stored in HDF format

4. checkTimeAxis(dataframe,path_to_summary_txt_file)- This function calculate metrics of the raw data samples: monotonic/mean dt/jumps in ms

5. checkTimeAxisOf2DataFrames(dataframe1, dataframe2,path_to_summary_txt_file)- This function calculate metrics of the raw data samples for how much 2 dataframes are synces: monotonic/mean dt/jumps in ms

6. mySyncingFunction(dataframe1,dataframe2, path_to_summary_txt_file,path_to_dataframes_hdf_files)- This function syncs both hand sensor measurements and back sensor measurements to each other and also linear acceleration to quaternion measurements based on the unix time
