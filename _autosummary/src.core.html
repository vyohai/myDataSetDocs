src.core

This module uses all the “helpers” modules to create ead a fullpartial ready-to-use dataset:

create the following directory tree:

/myDataSet/
/rawData/
/<subjectNmae>/
/compressedRawFiles/

/”adie_C3B_2021-12-13T15.52.14.492_C3B38AD76217_LinearAcceleration.h5” /…

/compressedSyncedFiles/

/”adie_C3B_2021-12-13T15.52.14.492_C3B38AD76217_LinearAcceleration.h5” /…

/sensoresSyncedFiles/

/adie_C3B_2021-12-13T15.52.14.492_C3B38AD76217_LinearAcceleration.h5” /…

/summary/

/<sensor_name1>_LA.txt /<sensor_name1>_Q.txt /<sensor_name1>_LA_Q.txt /both_LA_synced.txt /both_Q_synced.txt /…

Directories

  • /compressedRawFiles/ - this dir contain compressed format of the measurements dataframes(no preprocessing)

  • /compressedSyncedFiles/ - this dir contain compressed format of the measurements dataframes synced for each sensor

  • /sensoresSyncedFiles/ - this dir contain compressed format of the measurements dataframes synced between sensors

  • /summary/ - this dir contain the summary of the timestemps validation summery for each file and the syncing validation between the files timestamp

Summary txt files

  • /<sensor_name1>_LA.txt - sensor1, LA file timestamp validation summary

  • /<sensor_name1>_Q.txt - sensor1, Q file timestamp validation summary

  • /<sensor_name1>_LA_Q.txt - sensor1, LA and Q files syncing validation summary

  • /both_LA_synced.txt - LA files for both sensors syncing validation summary

  • /both_Q_synced.txt - Q files for both sensors syncing validation summary
